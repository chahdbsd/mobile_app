## Météo Mobile App

Application mobile qui affiche la météo actuelle et les prévisions à partir d’une API météo.
L’utilisateur utilise sa géolocalisation pour obtenir des informations météo fiables.

### Fonctionnalités
- Recherche par ville
- Météo actuelle 
- Affichage : température, état du ciel, humidité, vent 
- Gestion du chargement et des erreurs (API / réseau)
- Géolocalisation 

### Technologies
 React Native + Expo + API REST + JSON
